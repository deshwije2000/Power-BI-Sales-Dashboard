# Power BI Sales Performance Dashboard

### An interactive dashboard for analyzing retail superstore sales data to uncover business insights.
---

## üìù Project Overview

This project is a comprehensive sales analysis dashboard built from the ground up using Microsoft Power BI. 
The goal was to take a raw dataset of superstore sales, transform it into a usable format, and create a user-friendly dashboard to visualize key performance indicators (KPIs) and identify trends. 
This allows for data-driven decision-making regarding sales strategy, profitability, and customer segmentation.

## üõ†Ô∏è Tech Stack & Tools

* **Microsoft Power BI**: The core tool used for data visualization and dashboard creation.
* **Power Query**: Used for the Extract, Transform, Load (ETL) process to clean and prepare the data for analysis.
* **DAX (Data Analysis Expressions)**: Used to create custom calculations and measures, such as `Profit Margin`, to add deeper analytical value.

---

## ‚ú® Key Features & Analysis

This dashboard provides insights into several key areas of the business:

* **Executive KPI Summary**: At-a-glance cards displaying high-level metrics:
    * **Total Sales**: ~$2.30M
    * **Total Profit**: ~$286.4K
    * **Total Quantity Sold**: ~38K
    * **Average Discount**: 15.6%
    * **Profit Margin**: A custom DAX measure showing overall profitability.

* **Time Series Analysis**: A line chart tracks **Sales and Profit** over time, helping to identify seasonal trends and performance patterns.

* **Geospatial Analysis**: A map visualizes **Profit by State**, making it easy to identify the most and least profitable regions.

* **Product Performance**: A drill-down bar chart shows **Sales by Product Category and Sub-Category**, highlighting top-performing product lines.

* **Customer Segmentation**: A donut chart breaks down **Sales by Customer Segment** (Consumer, Corporate, Home Office) to understand the most valuable customer groups.
